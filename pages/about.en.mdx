# About this project

This project was started as part of the **Seminar Course: Astronomy and Astrophyics** and was intended to provide an introduction to **Fourier Analysis** for students.

The information was collected, implemented and written by _us_ (two students in the 13th grade).
We welcome improvements, feedback and notifications of possible errors.

## Content

- The content is based on many different sources and self-made explanations.

Specific references can be found at [references](./references)
